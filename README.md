# Transport BATI BRICO V1

Calculateur de zone transport utilisant OpenStreetMap Nominatim + OSRM public.

Fonctions :
- géocodage adresse client
- calcul trajet depuis Ambert et Arlanc
- choix dépôt le moins coûteux
- ajout 30 min chargement + 30 min déchargement
- coût = km A/R x 0,77 + temps total x 16 €/h
- zone conseillée selon barème HT
